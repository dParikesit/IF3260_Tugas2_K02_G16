Help

  • (Optional) Import objek menggunakan button bagian loader
  • Pilih opsi di draw untuk memulai menggambar
  • Pencet canvas untuk menggambar objek
    1. POLYGON: Klik kanan pada saat membuat titik terakhir, untuk berhenti membuat polygon
  • Untuk mengubah warna vertex dan memindahkan vertex, pilih point di sidebar kanan kemudian pilih opsi di bagian Style
  • Untuk melakukan translasi / rotasi objek, pilih objek di sidebar kanan kemudian geser slider di bagian Transformasi
  • Untuk menghapus vertex pada polygon, klik delete button pada sidebar kanan. Penghapusan hanya dapat dilakukan untuk polygon dengan vertex lebih dari 4
  • Untuk menambah vertex dengan posisi random pada polygon, klik add button pada sidebar kanan.
  • (Optional) Export menggunakan button export di bagian loader
